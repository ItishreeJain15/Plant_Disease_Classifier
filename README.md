# Plant_Disease_Classifier
**📌 Overview**\
AI-powered deep learning system that classifies medicinal plants from images\
**🚀 Features**\
🌱 Image-based medicinal plant identification\
🤖 Deep learning (CNN) model for classification\
📊 Trained on publicly available datasets\
📚 Displays plant name \
🖥️ Easy to run in Colab/Jupyter Notebook\
**🛠️ Tech Stack**\
Python\
TensorFlow / Keras\
NumPy, Pandas\
Matplotlib\
**📥 Dataset** :- Kaggle (Medicinal Plant Dataset)\
**▶️ Usage**\
Open Jupyter Notebook\
Run all cells step-by-step\
Upload an image for prediction\
Model will classify the plant\
**📸 Sample Output**\
Predicted Plant Name\

Educational purposes\
